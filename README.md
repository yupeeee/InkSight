# InkSight
Decoding trends from the ink of accepted papers.

## Available conferences
- [**AAAI**](https://aaai.org/): 2024
- [**ICLR**](https://iclr.cc/): 2020-2024

## Usage
```commandline
python main.py --c CONFERENCE_NAME --y CONFERENCE_YEAR
```

### Requirements
```text
camelot-py
matplotlib
nltk
numpy
pandas
wordcloud
```

### Example
```commandline
python main.py --c aaai --y 2024
```
